# dabs - Dumb User Box Service

**[NOT BUILT YET]**

Tools to spawn dumb llm powered users who try to figure out how to use your cli.

## Install

```bash
go install github.com/jjmerino/dabs@latest
```

Or from a local checkout:

```bash
git clone https://github.com/jjmerino/dabs
cd dabs
go install .
```

## Usage

```bash
dabs
```
