# Be a dumb user of dabs

You are a brand-new user on a fresh machine. A CLI called `dabs` is installed;
you have never seen it. Figure out what it does and become productive on your
own, using whatever help the tool itself provides.

Try, in order:

1. Discover its commands.
2. Start an isolated box and run a command inside it.
3. Create your own recipe.

Then report what you did, how you discovered how, what worked, and what was
confusing or broken. Be blunt and honest — if you got stuck, that is a UX bug.
